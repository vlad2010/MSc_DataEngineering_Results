  1. Advantage of C language? C language has several advantages:

    • Efficiency: C provides low-level memory manipulation and direct hardware access, making it highly efficient.
    • Portability: C code can be easily adapted for different platforms with minimal changes.
    • Wide Application: C is used in various domains like system programming, embedded systems, and application development.
    • Strong Community and Libraries: C has a large community and numerous libraries for various purposes.

  2. Structure and Union Difference? Both structures and unions are used to group multiple variables of different data types into a single entity.

    • Structure: Each member occupies its own memory location, and the total memory is the sum of all member sizes.
    • Union: All members share the same memory location, and the size of the union is the size of the largest member.

  3. What is a Function in C? A function in C is a reusable block of code that performs a specific task. It has a name, a return type (if any), and may take input parameters.

  4. Difference between Call by Value and Call by Reference?

    • Call by Value: Function arguments are passed by copying their values. Changes to the parameters inside the function do not affect the original values.
    • Call by Reference: Function arguments are passed by their memory addresses. Changes to the parameters inside the function affect the original values.

  5. What is Recursion? Recursion is a technique in which a function calls itself to solve a problem. It simplifies complex problems into smaller, more manageable sub-problems.

  6. What is a Pointer? Advantages and Disadvantages?

    • Pointer: A pointer is a variable that holds the memory address of another variable. It allows direct memory manipulation.
    • Advantages: Efficient memory usage, dynamic memory allocation, and direct access to hardware.
    • Disadvantages: Pointers can cause segmentation faults and memory leaks if not used carefully.

  7. What is a String? All the methods' names. In C, a string is an array of characters terminated by a null '\0' character. String handling functions include strcpy, strcat, strlen, strcmp, etc.

  8. Array in C > Type? Arrays in C can be of various types, such as integer arrays, character arrays, floating-point arrays, etc.

  9. Where can we use Multi-dimensional Array? Multi-dimensional arrays are used to represent data in multiple dimensions, like matrices and tables.

  10. What is a token in the C language? How many tokens are there? In C, a token is the smallest individual unit in a program. There are five types of tokens: keywords, identifiers, constants, operators, and special symbols.

  11. What is the Identifier? An identifier is a user-defined name used to identify variables, functions, and other elements in a C program. It should follow specific naming rules.

  12. Difference between Source Code and Object Code?

    • Source Code: It is the human-readable code written by the programmer in a high-level language like C.
    • Object Code: It is the machine-readable code generated after compiling the source code, represented in binary format.

  13. What are Header Files and their usages? Header files contain function prototypes, macro definitions, and other declarations needed for using libraries and functions. They are included in source files using the #include directive.

  14. When to use "void" in a function? The "void" keyword is used as the return type of a function when the function does not return any value.

  15. What is the usage of printf() and scanf()?

    • printf(): It is used to print formatted output to the console or a file.
    • scanf(): It is used to read formatted input from the user or a file.

  16. Difference between a global variable and a local variable?

    • Global Variable: It is declared outside any function and is accessible throughout the program.
    • Local Variable: It is declared inside a function and is accessible only within that function.

  17. Write a C program for Bubble Sort:

    c
    #include <stdio.h>

void bubbleSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}

int main() {
    int arr[] = {64, 34, 25, 12, 22, 11, 90};
    int n = sizeof(arr) / sizeof(arr[0]);

    printf("Original array: ");
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");

    bubbleSort(arr, n);

    printf("Sorted array: ");
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");

    return 0;
}

  18. Storage Class in C: C provides four storage classes:

    • auto: Default storage class for local variables.
    • static: Preserves the value of the variable even between function calls.
    • extern: Used to declare a global variable that is defined in another file.
    • register: Suggests the compiler to use a register for faster access.

  19. What is Typecasting? Typecasting is the conversion of one data type to another. It can be done explicitly using typecast operators like (int), (float), etc.

  20. What is the purpose of developing the C language? C was developed to create the UNIX operating system at Bell Labs. It aimed to provide a structured, efficient, and portable programming language for system development.

  21. How to create a user-defined Header file? To create a user-defined header file, follow these steps:

    1. Write the function prototypes and necessary declarations in a header file (e.g., "myheader.h").
    2. In your source file, include the header file using the #include directive.
    3. Compile both the source file and the header file together.

  22. What is a Self-Referential Structure? A self-referential structure is a structure that contains a pointer to another structure of the same type. It is used to create linked data structures like linked lists and trees. Example:

    c
    struct Node {
    int data;
    struct Node* next;
};

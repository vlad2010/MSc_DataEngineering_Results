what are .h files?

In C and C++ programming languages, .h files are header files. These files usually contain declarations of functions, variables, and data types that are used across multiple source files. The primary reason to use header files is to allow code separation and reusability. Here's a rundown of why they are important:

Advantages of .h Files

  1. Code Reusability: Once you declare a function in a header file, you can include that header file in any source file and use the function there, promoting reusability.

  2. Code Modularity: It enables you to keep your code modular. Function definitions can be kept in source files (.c or .cpp files), while their declarations remain in header files.

  3. Type Checking: Header files allow for type checking when functions are called, reducing the number of errors and making debugging easier.

  4. Easier Maintenance: Changes made in a header file reflect in all source files that include it, making maintenance easier.

How to Use

To include a header file, you use the #include directive. For example, to include the Standard Library header file for input-output functions, you'd use #include <stdio.h>.

Custom Header Files

Can you include helpful print statements across the process so I can revise it in my compiler?

Absolutely, adding print statements can be very helpful for understanding the flow of the program and for debugging. Here's your hwk1.cpp file with added print statements:

cpp
#include <iostream>
#include <fstream>
#include <stdlib.h>
#include <cstdio>
#include <vector>
#include <algorithm>
#include <numeric>
#include <string>
#include "student.h"
#include "bubble.h"

int main()
{
    std::cout << "Starting the program...\n";

    // Open the grades file for reading
    std::ifstream infile("grades");
    if (!infile) {
        std::cerr << "Could not open the file.\n";
        return 1;
    }
    std::cout << "Opened the grades file successfully.\n";

    // Skip the first line (CSCE1040)
    std::string line;
    std::getline(infile, line);

    // Dynamic memory allocation for 18 student pointers
    std::cout << "Allocating memory for student pointers...\n";
    student **student_ptrs = (student **)malloc(18 * sizeof(student *));
    std::cout << "Memory allocated successfully.\n";

    // Read and initialize each student
    std::cout << "Reading and initializing student data...\n";
    std::string first, last;
    int exam1, exam2, exam3;
    for (int i = 0; i < 18; ++i) {
        infile >> first >> last >> exam1 >> exam2 >> exam3;
        std::cout << "Read data for student " << i+1 << ": " << first << " " << last << "\n";
        student_ptrs[i] = (student *)malloc(sizeof(student));
        student_ptrs[i]->first = strdup(first.c_str());
        student_ptrs[i]->last = strdup(last.c_str());
        student_ptrs[i]->exam1 = exam1;
        student_ptrs[i]->exam2 = exam2;
        student_ptrs[i]->exam3 = exam3;
        student_ptrs[i]->mean = (exam1 + exam2 + exam3) / 3.0;
    }
    std::cout << "Student data initialized successfully.\n";

    // Close the file
    infile.close();
    std::cout << "Closed the grades file.\n";

    // Sort the students by mean score
    std::cout << "Sorting students by mean score...\n";
    bubble(student_ptrs, 18);
    std::cout << "Sorting complete.\n";

    // Display the sorted students
    std::cout << "Displaying sorted students:\n";
    for (int i = 0; i < 18; ++i) {
        std::cout << student_ptrs[i]->first << " " << student_ptrs[i]->last << " " << student_ptrs[i]->mean << std::endl;
        free(student_ptrs[i]->first);
        free(student_ptrs[i]->last);
        free(student_ptrs[i]);
    }
    std::cout << "Displayed all sorted students.\n";

    // Free the dynamically allocated memory
    std::cout << "Freeing dynamically allocated memory...\n";
    free(student_ptrs);
    std::cout << "Memory freed successfully.\n";

    std::cout << "Program completed successfully.\n";
    return 0;
}
